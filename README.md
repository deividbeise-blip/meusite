- Cursor personalizado animado
- Animações de scroll com Intersection Observer
- Barras de habilidade que se preenchem ao rolar a página
- Cards de projetos clicáveis linkados ao GitHub
- Design responsivo para mobile e desktop
- Tema escuro com identidade visual única
